喵记知识库设计规范

喵记知识库是一款面向开发者和知识工作者的本地优先笔记工具，兼顾深度专注与趣味陪伴两大核心体验。它将双向链接的知识管理能力与 AI 驱动的猫咪宠物伴侣巧妙融合，打造「沉浸写作 + 智能助手 + 情绪价值」三位一体的桌面端应用。

核心架构

应用采用非对称四栏布局，在保障编辑区最大沉浸空间的同时，让导航与 AI 助手随时可达。设计理念源自 [[毛玻璃拟态设计原则]]

毛玻璃拟态设计原则
一种现代 UI 设计趋势，通过磨砂玻璃效果、半透明背景和柔和模糊来营造界面的深度感与层次感，让元素之间产生自然的视觉分隔...
，打造无硬边框的纯净分层界面。

数据采用 Markdown 文件与轻量级 SQLite 数据库的组合方式存储在本地，确保完全离线可用和用户数据主权。这一设计遵循了 [[本地优先软件宣言]]

本地优先软件宣言
一种软件设计理念：数据主要在用户本地设备上存储和处理，云端同步仅作为辅助机制，确保用户对自己数据拥有完全的控制权...
的核心原则。

AI 宠物伴侣系统

AI 猫咪伴侣驻留在界面的第四栏，承担多重角色：

  • 语境感知辅助 - 猫咪会观察你的写作上下文，提供内联建议、内容摘要和组织优化提示。
  • 情绪陪伴互动 - 通过生动的动画响应和趣味互动，在长时间写作中营造陪伴感。
  • 文档操作动效 - 将文档拖拽给猫咪即可归档，猫咪会跃起张嘴「吃掉」文档，配合令人满足的吞咽动画。
AI 建议
建议补充关于知识图谱可视化算法的章节。力导向布局可以使用 D3.js 实现，或采用更轻量的 ngraph.forcelayout 库以获得更好的实时渲染性能。同时建议为图谱节点添加颜色编码以区分不同类型的笔记。
「最好的工具，是让人感觉它是思维的自然延伸。喵记的目标是：在你心流状态时隐于无形，在你需要灵感时温柔出现。」

技术栈选型

应用使用 Tauri 2.0 构建桌面外壳，React 18 负责 UI 层，Three.js 用于 3D 猫咪渲染。状态管理选用 Zustand，因其 API 精简且对 TypeScript 支持优秀。

知识图谱采用自研的力导向布局算法，针对最多 500 个节点的实时渲染进行了优化。Markdown 编辑器底层基于 [[ProseMirror 架构设计]]

ProseMirror 架构设计
一套基于函数式、不可变文档模型构建富文本编辑器的工具集，拥有强大的插件系统，支持协同编辑和自定义节点类型...
框架，并通过自定义插件扩展了双向链接解析和 AI 内联建议功能。
